```
G 弦（最上面）= G
C 弦 = C
E 弦 = E
```

## 大调音阶公式

> 全全半全全全半（从 do 开始按这个距离排列出 7 个音）
> 全音 = 往右 2 品，半音 = 往右 1 品

## 如何找任意 Key 的位置

```
1. 先在四根弦上找到 do 的位置（Key 的名字）
2. 按"全全半全全全半"的距离往后推
3. 全音 = 往右 2 品，半音 = 往右 1 品
```

## 常用 Key 排行

| 排名 | Key | 升降号 | 原因 |
|------|-----|--------|------|
| 1    | C   | 无     | 最简单，没有升降号 |
| 2    | G   | 1个#   | 只有一个 F# |
| 3    | F   | 1个b   | 只有一个 Bb |
| 4    | D   | 2个#   | 吉他/尤克里里常用 |
| 5    | A   | 3个#   | 常见于流行歌 |
| 6    | Bb  | 2个b   | 管乐器常用 |
