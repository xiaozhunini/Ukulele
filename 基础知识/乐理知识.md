# 什么是Key
> 一首歌的“基准音”，决定了整首歌用什么音来弹奏
>       		Key 就像一个"颜色基调"——如果 Key 是 C 大调，整首歌就主要用 C D E F G A B 这 7 个音来构成，就像一幅画定好了暖色调。
> 就是一首歌用什么大调弹唱

# key  对应 音

| Key  | do   | re   | mi   | fa   | sol  | la   | si   |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| C    | C    | D    | E    | F    | G    | A    | B    |
| C#   | C#   | D#   | F    | F#   | G#   | A#   | C    |
| D    | D    | E    | F#   | G    | A    | B    | C#   |
| Eb   | Eb   | F    | G    | Ab   | Bb   | C    | D    |
| E    | E    | F#   | G#   | A    | B    | C#   | D#   |
| F    | F    | G    | A    | Bb   | C    | D    | E    |
| F#   | F#   | G#   | A#   | B    | C#   | D#   | F    |
| G    | G    | A    | B    | C    | D    | E    | F#   |
| Ab   | Ab   | Bb   | C    | Db   | Eb   | F    | G    |
| A    | A    | B    | C#   | D    | E    | F#   | G#   |
| Bb   | Bb   | C    | D    | Eb   | F    | G    | A    |
| B    | B    | C#   | D#   | E    | F#   | G#   | A#   |

> 一根琴弦上的排列方式，每根琴弦只是起点不同
>
> C → C# → D → D# → E → F → F# → G → G# → A → A# → B → C
>   半    全   半    全  半   半    全   半    全   半    全
>
> 注意：
>
> - **C# = Db**（同一个音，两种叫法）
> - **D# = Eb**（同一个音）
> - **F# = Gb**（同一个音）
> - **G# = Ab**（同一个音）
> - **A# = Bb**（同一个音)
>
> ```
> do re mi fa sol la si = Key 规定的 7 个"位置标签"
> ```



# 什么是全音半音

```
弹两个相邻的音（半音）：比如 C 和 C#
→ 听起来很"紧"、"刺"、不和谐

弹隔一个品的音（全音）：比如 C 和 D
→ 听起来比较"顺"、"舒服"
```

## 用钢琴理解最清楚



```
白键：C   D   E   F   G   A   B   C
黑键： C#  D#      F#  G#  A#

      C→D = 全音（中间有黑键 C#）
      D→E = 全音（中间有黑键 D#）
      E→F = 半音（中间没有黑键！）
      F→G = 全音（中间有黑键 F#）
      G→A = 全音（中间有黑键 G#）
      A→B = 全音（中间有黑键 A#）
      B→C = 半音（中间没有黑键！）
```

## 记忆口诀



```
E-F 半音，B-C 半音，其他都是全音
```

## 为什么 E-F 和 B-C 之间没有升降音

这是历史原因，音乐系统就是这样设计的。就像一周有 7 天，没有为什么，就是这么定的。记住这个规律就行：



```
C  D  E  F  G  A  B
 全 全 半 全 全 全 半
      ↑        ↑
    E-F      B-C
   天然半音  天然半音
```

## 尤克里里（记忆）

半音 = 隔 1 个品格
全音 = 隔 2 个品格

## 为什么大调公式是"全全半全全全半"

```
do re mi fa sol la si do

如果全是全音：C D E F# G# A# C
→ 听起来怪怪的，没有"do re mi"的感觉

如果按"全全半全全全半"：C D E F G A B C
→ 听起来就是你熟悉的"do re mi fa sol la si"
```
